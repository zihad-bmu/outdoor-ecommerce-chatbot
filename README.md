## 🚀 Features

### 1️⃣ Order Tracking
Uses exact mock order logic:
- Order #111 → Shipped, arriving tomorrow
- Order #222 → Processing, ships in 24 hours
- Order #333 → Delivered (with follow-up)
- Any other number → invalid order

### 2️⃣ Returns & Exchanges
Displays:
- 30-day returns
- Items must be unused
- Original packaging required
- Returns link included

### 3️⃣ Product Recommendations
- Asks 2 clarifying questions
- Recommends product category based on answers

### 4️⃣ Human Handoff
- Triggered by explicit request
- Triggered after repeated fallback
- Simulated Live Agent state
- User can return to main menu anytime

### 5️⃣ Shipping Information
- Standard shipping: 3-5 business days
- Expedited shipping: 1-2 business days
